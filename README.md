## 📖 Overview

This project is a simple tribute webpage created as part of my **Internet Programming coursework**. The objective was to demonstrate the use of fundamental **HTML elements** while applying basic **CSS styling** to produce a clean, responsive, and visually appealing webpage.

The webpage is dedicated to an individual I admire and showcases structured content using semantic HTML and page layout techniques.

---

## ✨ Features

- 📄 Well-structured HTML document
- 🎨 Basic CSS styling for improved appearance
- 📱 Responsive page layout
- 🏷️ Proper use of HTML tags and semantic elements
- 🖼️ Images and formatted content
- 🔗 Hyperlinks and navigation elements
- 💻 Beginner-friendly frontend implementation
